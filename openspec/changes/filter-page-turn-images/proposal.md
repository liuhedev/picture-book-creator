# Change: 过滤翻页图片

## Why
在质检部分，用户需要手动筛选图片。有些图片包含翻页动画的中间帧，这些图片通常显示的是翻页过程中的模糊状态或过渡效果，不是最终需要的静态内容。用户希望在质检界面中能够自动识别并过滤掉这些翻页图片，减少手动筛选的工作量。

## What Changes
- 新增翻页图片检测功能（在图片提取阶段或质检阶段检测）
- 在质检界面中自动隐藏或标记翻页图片
- 提供过滤选项，允许用户选择是否显示翻页图片
- 更新图片统计信息，显示翻页图片数量

## Impact
- 修改能力：`manual-review`（质检界面）
- 可能修改能力：`image-quality-filter`（如果在提取阶段检测）
- 影响代码：前端界面、可能的后端检测逻辑
- 影响：质检流程中自动过滤翻页图片，提升用户体验
